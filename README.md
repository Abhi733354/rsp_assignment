# React + TypeScript + Vite

##Overview:
This project visualizes the Retail Selling Price (RSP) of Petrol and Diesel in Indian Metro Cities, based on the dataset made available by the National Data and Analytics Platform (NDAP), NITI Aayog.

## **The app allows users to select**

- A Metro City
- A Fuel Type
- A Calendar Year
and view a bar chart showing the monthly average RSP for that selection.

## **Tech Stack**

TypeScript
Vite (React + TypeScript template)
Mantine UI (for dropdowns and layout)
Apache ECharts (for data visualization)
No helper libraries used — no Bootstrap, jQuery, Lodash, etc.

## **Features**

✅ Load and parse CSV data for petrol & diesel RSPs.
✅ Clean dropdowns for selecting City, Fuel, and Year.
✅ Interactive bar chart rendered using Apache ECharts.
✅ Responsive and lightweight frontend.
✅ Missing or invalid values are treated as 0.
✅ Built entirely with TypeScript (no JavaScript).
